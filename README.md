# [ICT-울산_정희섭]My Diary


## 소개

ICT 교육과정의 마지막인 프로젝트 입니다.


## 프로젝트 주제

각종 도서와 독서감상문을 공유하는 웹 사이트라는 주제로 프로젝트를 작성하였습니다.


## 목표

- JS 대신 TS를 사용함으로써 도전의식 부여
- 그외 익숙하지 않은 다양한 기술(MongoDB, 


## 구현해야할 기능

- 자기소개 페이지 만들기
- 게시판 만들기(CRUD 구현)
- 페이지네이션 기능 만들기
